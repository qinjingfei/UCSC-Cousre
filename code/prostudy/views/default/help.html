  • How can I use Pro Study?
    • Create an Account
    • Login
    • Input a list of tasks that you normally do
    • Categorize each task according to its usage
    • Set the ideal time that you would spend on each task
    • Once you have your list done, select a task and start the timer
    • Once you are done with the task or feel you want to switch, click the stop button
    • Repeat the same process until you are done with your work
